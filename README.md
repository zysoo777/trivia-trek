# Trivia Trek

Trivia Trek is a simple quiz game built with React and Vite. Users can choose a difficulty level and category, answer multiple-choice questions, and see their final score, percentage, message, and rank at the end of the game.

## Features

- Choose quiz difficulty
- Choose quiz category
- Answer multiple-choice questions
- See score, percentage, and rank at the end
- Quit the quiz at any time
- Replay the game

## Built With

- React
- Vite
- CSS
- Open Trivia DB API

## How to Run Locally

1. Open the project folder in your code editor
2. Open a terminal in the project folder
3. Run `npm install`
4. Run `npm run dev`
5. Open the local link shown in the terminal

## Notes

This project uses the Open Trivia DB API to load quiz questions.
Questions are fetched dynamically based on the selected difficulty and category.
